# Pari o Dispari - Java

Questo progetto in **Java** permette di verificare se un numero inserito dall'utente è **pari** o **dispari**.  
Il progetto include una funzione `isEven()` che può essere usata per determinare il risultato.

## Funzionalità

- Inserimento di un numero da tastiera.
- Controllo se il numero è **pari** o **dispari**.
- Restituisce il risultato come:
  - Messaggio stampato su console
  - Booleano `true` (pari) / `false` (dispari)
